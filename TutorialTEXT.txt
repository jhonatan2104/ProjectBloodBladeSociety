BLOOD BLADE SOCIETY

        BLOOD BLADE SOCIETY - BBS É UM GAME ONDE VOCÊ TEM A OPORTUNIDADE DE ESCOLHER ENTRE ALGUNS HERÓIS DA CULTURA POP
        E BATALHAR CONTRA A UMA INTELIGENCIA DE GERENCIAMENTO DE DADOS. VOCÊ É MAIS FORTE QUE A MÁQUINA? ISSO NÓS VAMOS
        VER...

        CADA CAMPEÃO É PERSONALIZADO E DETEM CARACTERISTICAS DIVERSAS, COM PERSONALIDADE E ATRIBUTOS DISTINTOS.

    PLAYER:
    - LIFE
    - MANA

        CADA PLAYER POSSUI EQUIPAMENTOS TAMBÉM

    - ESPADA : CADA ESPADA POSSUI TRÊS ATAQUES
    - ARMADURA : CADA ARMADURA POSSUI DEFESA FÍSICA, DEFESA MÁGICA E LATÊNCIA

    OBS: LATÊNCIA É UM NÚMERO DE 0 A 9, NO QUAL DETERMINA A POSSIBILIDADE DE ERRO. SE UM EQUIPAMENTO POSSUI LATÊNCIA 4,
    SIGNIFICA QUE, NO DADO ROLADO, SE O NÚMERO QUE CAIR FOR MAIOR OU IGUAL A 4, O EQUIPAMENTO( ARMADURA E ATAQUE )  F
    OI EFETIVO.

    -- ATAQUES:               --- DANO MÁGICO
    --- DANO FÍSICO           --- MANA PARA USO
    --- LATÊNCIA

    ITENS:

        OS ITENS SÃO FERRAMENTAS PRIMORDIAIS PARA A VITÓRIA. ELES PODEM ALTERAR MUITAS VARIÁVEIS DE JOGO. COMO:

     -Life  -Defesa Magica  -Latência Ataque  -Dano Magico
     -Mana  -Defesa Fisica  -Latência Defesa  -Dano Fisico

        OS ITENS POSSUI COMO ATRIBUTO:

        -- VALOR DE COMPRA ( MOEDAS )
        -- QUANTIDADE
    OBS: MOEDAS SÃO CONSEGUIDAS TODO INÍCIO DE PARTIDADA E VAI SE ACUMULANDO AO PASSO QUE SEUS ATAQUE VÃO SENDO EFETIVOS
    DE ACORDO COM O SEGUINTE CALCULO:

    	base = 25
    	extraType = 5 if ESTATUS DO ATAQUE == "FALHO" else 25 if ESTATUS DO ATAQUE == "ATAQUE NORMAL" else 55 if ESTATUS DO
    	ATAQUE == "ATAQUE CRÍTICO" else 0
    	extraDano = 5 if dano < 1000 else 7 if dano < 2000 else 10 if dano < 3000 else 15 if dano < 4000 else 30 if dano <
    	5000 else 45

    		MOEDAS GANHAS POR RODADA = base + extraType + extraDano

    OBS: A MANA É USADA PARA REALIZAR OS ATAQUES. ELA TAMBÉ É RESTAURADA A CADA RODADA DE ACORDO COM O SEGUINTE CALCULO:

    		MANA RESTAURADA = int(1000 * (100 / dano)) if dano >= 400 else 300

CONFIG TECLADO:

Q - ATAQUE 1
W - ATAQUE 2
E - ATAQUE 3
R - MODO DEFENSIVO

A - DICA ATAQUE 1
S - DICA ATAQUE 2
D - DICA ATAQUE 3

C - INVENTÁRIO
V - LOGA DE ITENS

J - DADOS DA ESPADA
K - DADOS DA ARMADURA

ENTER - CONTINUAR DEPOIS DE REALIZAR O ATAQUE
